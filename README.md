# 📊 Data Analyst Assignment – PlatinumRx

## 📌 Overview

This project demonstrates core data analysis skills including SQL, Excel, and Python. The assignment is divided into three major parts:

* SQL (Database queries)
* Spreadsheets (Excel analysis)
* Python (Basic programming tasks)

## 🧠 SQL Tasks

### Hotel Management System

* Retrieved last booked room for each user
* Calculated total billing for November 2021
* Filtered bills greater than 1000 in October 2021
* Identified most and least ordered items per month
* Found customers with second highest bill per month

### Clinic Management System

* Calculated revenue by sales channel
* Identified top 10 customers
* Computed monthly revenue, expenses, and profit
* Found most profitable clinic per city
* Found second least profitable clinic per state
